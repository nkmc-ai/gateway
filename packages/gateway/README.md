# @nkmc/gateway

> The gateway of internet for all agents. NaKaMiChi - the middle path, the right path.

Core gateway library — credential vault, service registry, federation, CLI proxy, and tunnel support.

## Part of the [nkmc gateway](https://github.com/nkmc-ai/gateway)

```bash
npm install @nkmc/gateway
```

## What it does

- **Credential Vault** — AES-GCM encrypted storage, Pool + BYOK model
- **Service Registry** — Register APIs via OpenAPI auto-discovery or skill.md
- **Federation** — Peer gateways lend credentials via lending rules + x402
- **CLI Proxy** — Execute CLI tools with injected credentials
- **Tunnel** — Cloudflare Tunnel integration for NAT traversal + discovery
